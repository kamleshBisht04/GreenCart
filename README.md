🛒 GreenCart – MERN Stack Grocery App
GreenCart is a full-stack MERN (MongoDB, Express, React, Node.js) grocery e-commerce application with modern features like authentication, cart system, payment integration, image upload, and category management.

🚀 Live Demo
(Add your deployed links here if available)
project link: [https://your-frontend-url.com](https://greencart-eta-sage.vercel.app/)


✨ Features
🔐 User Authentication (Login / Signup)
🍏 Grocery Product Listing
📂 Dynamic Categories Section
🛒 Add to Cart & Remove from Cart
💳 Razorpay Payment Integration
🍪 Cookie-based Authentication System
☁️ Image Upload using Cloudinary
📦 File Upload using Multer
🔍 Product Filtering by Category
📱 Fully Responsive UI (Mobile + Desktop)
⚡ Fast and Optimized React Frontend

🧰 Tech Stack
Frontend:
React.js
Tailwind CSS (v4)
React Router
Context API

Backend:
Node.js
Express.js
MongoDB + Mongoose
JWT Authentication
Cookies (HTTP-only)
Services & Tools:
Razorpay (Payment Gateway)
Cloudinary (Image Hosting)
Multer (File Upload Handling)
dotenv (Environment Variables)
